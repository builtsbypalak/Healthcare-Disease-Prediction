# Healthcare Disease Prediction using Machine Learning

This project predicts diabetes using patient medical data and machine learning algorithms.

## Models Used
- Logistic Regression
- Random Forest

## Steps
- Data Cleaning
- Exploratory Data Analysis
- Feature Scaling
- Model Training
- Model Evaluation
- Prediction System

## Accuracy
~80%

## Tools
Python, Pandas, NumPy, Scikit-learn, Seaborn, Matplotlib
